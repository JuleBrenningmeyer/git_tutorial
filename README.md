# Beginner Git Tutorial

Welcome to the Beginner Git Tutorial! This repository is designed to help you learn the fundamental concepts and commands of Git and GitHub through simple, step-by-step guides. 

## 📚 Table of Contents

We recommend following these guides in order if you are completely new to Git:

1. **[Creating a Repository](CreateRepo.md)**
   Learn how to initialize a brand new Git project on your computer and connect it to GitHub.
   
2. **[Staging and Committing Files](StagingFiles.md)**
   Understand the core two-step saving process in Git: adding to the Staging Area and Committing.

3. **[Creating and Switching Branches](CreateBranch.md)**
   Learn how to create isolated "workspaces" for your new feature ideas, and how to safely merge or delete them when you're finished.

4. **[Cloning and Pulling](CloneAndPull.md)**
   Find out how to download existing projects from GitHub to your computer and keep your local files updated with the latest changes.

5. **[Addressing Common Errors](ErrorHandling.md)**
   Everyone makes mistakes! Learn essential commands to figure out what went wrong and how to safely fix it (like using `git restore` and amending commits).

## 🚀 Getting Started

If you are ready to begin, click on the first lesson: **[Creating a Repository](CreateRepo.md)**!
